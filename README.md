# CppGames
1) Тетрис - Данная игра является реализацией кода из https://github.com/jsyqrt/games/tree/master/01%20Tetris, только с использованием ООП.
2) Doodle Jump - Игра является продолжением цикла игр из https://github.com/jsyqrt/games, но на этот раз без опоры на предложенные исходники (то есть, это моя личная реализация).
3) Arcanoid - Продолжаем реализовывать игры из цикла. Так же без опоры на исходники, однако на этот раз получилось довольно громоздко. Стоит задуматься о целесообразности использования большого количества классов (Довольно однотипных, к слову).
